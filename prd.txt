## Overview

Reclaimer is a mobile application that connects individual users to nearby automated recycling kiosks for e-waste, textiles, and beverage containers. The app gamifies recycling through instant point rewards and redeemable incentives, while simultaneously solving operational challenges for recycling partners through data-driven insights and optimized collection logistics.

This PRD complements the executive product document by detailing the core user experience, feature specifications, and implementation priorities for the MVP phase.

## Core Product Features (MVP)

### Feature 1: Recycling Kiosk Finder

**What it does**: Interactive map showing nearby automated recycling machines with real-time availability and accepted materials

**User Experience**:

- Map view with kiosk pins color-coded by material type (blue = e-waste, green = textiles, orange = beverages)
- List view showing kiosks sorted by distance, with operating hours and accepted items
- Search/filter by material category or kiosk partner
- One-tap directions integration (Apple Maps / Google Maps)
- Ratings and user reviews (e.g., "Machine was full on Tuesday 3pm")

**Data Displayed**:

- Location, address, hours
- Currently accepting materials (real-time sync from kiosk)
- Partner name (e.g., "EcoPartner Recycling")
- User ratings and review count
- Distance from current location

**Technical**: Google Maps API integration, real-time kiosk status via partner API polling

### Feature 2: Deposit & Instant Rewards

**What it does**: Users scan item or kiosk QR code to log recycled materials and instantly earn points

**User Experience**:

- At kiosk: QR code visible on machine or receipt
- User opens app â†’ "Deposit Now" button â†’ Camera scans QR â†’ Item categorization screen appears
- User confirms item type and quantity (e.g., "1 smartphone" or "2kg textiles")
- Points credited instantly (haptic feedback, success animation, notification sound optional)
- Confirmation screen shows: points earned, total balance, and next reward milestone

**Points Structure**:

- Electronics: 10â€“50 points depending on size/weight (smartphone = 10, laptop = 50)
- Textiles: 5 points per kg

- Bulk bonus: 20% extra points if 5+ items in single deposit

**Technical**: Kiosk QR encoding, item-to-points mapping logic, secure point ledger

### Feature 3: Digital Wallet & Redemption
